# Glossário do Domínio (DOMAIN GLOSSARY)

> Definições da linguagem ubiqua utilizada no Leadgers Platform, para garantir alinhamento entre produto e engenharia.

| Termo                                | Definição                                                                                                                    |
| ------------------------------------ | ---------------------------------------------------------------------------------------------------------------------------- |
| **Multi-Tenant**                     | Arquitetura onde múltiplos clientes (startups) usam a mesma infraestrutura, mas têm seus dados rigidamente isolados.         |
| **Workspace (Tenant)**               | O ambiente fechado de uma Startup na plataforma.                                                                             |
| **Runway**                           | O tempo de vida (em meses) que a startup tem antes de o caixa zerar, calculado pelo `Saldo Atual / Net Burn Rate`.           |
| **Gross Burn**                       | Taxa de queima de caixa bruta; o total absoluto que a empresa gasta no mês.                                                  |
| **Net Burn**                         | Taxa líquida de queima; `Total de Saídas - Receitas Recorrentes (MRR)`.                                                      |
| **Cap Table**                        | Tabela de Capitalização. O mapeamento matemático de quem possui quais porcentagens (%) e ações da empresa.                   |
| **ESOP (Pool)**                      | Employee Stock Option Pool. Quantidade de ações retidas para distribuir aos funcionários e incentivá-los.                    |
| **Vesting**                          | Período e cronograma no qual o funcionário "ganha" o direito às suas opções de ESOP.                                         |
| **Cliff**                            | O período de carência (ex: 12 meses) onde, se o funcionário sair antes, perde todas as ações.                                |
| **OKR (Objectives and Key Results)** | Metodologia hierárquica de metas corporativas cascateadas.                                                                   |
| **Unit Economics**                   | Eficiência da unidade: CAC (Custo de Aquisição de Clientes) e LTV (Lifetime Value).                                          |
| **Health Score**                     | Índice composto de 0 a 100 gerado por IA/algoritmos para medir a saúde geral técnica, financeira e de compliance da startup. |
| **Weekly Digest**                    | Resumo semanal disparado toda 2ª feira, processado por LLM, destacando gargalos executivos.                                  |
